# Project Viral Scanner v1.0

## Objective

Viral Scanner v1.0 helps creators analyze why a viral video worked and convert those insights into original ideas.

This document defines the locked v1.0 product scope.

## Locked Roadmap Order

Gatekeeper AI roadmap order is fixed:

1. AI Idea Explorer
2. Content Creator v1.0
3. Viral Scanner v1.0
4. AI Production Studio
5. Publishing
6. AI Content Intelligence

No module reordering is allowed.
No extra modules are part of this scope.

## Product Purpose

Viral Scanner should answer:

"I found a viral video. Why did it work, and how can I create my own version?"

Viral Scanner is an analysis and idea adaptation module, not a media production engine.

## Locked Core Workflow

Paste Video Link / Manual Description
-> Analyze
-> Why It Works
-> Breakdown
-> Key Takeaways
-> Create My Version
-> Suggested Original Ideas
-> Continue to Content Creator

This workflow is required and should remain linear in v1.0.

## v1.0 Required Outputs

### Input

- Video link input
- Manual description input

### Analysis

- A concise structured analysis summary

### Why It Works

- Primary hook pattern
- Retention pattern
- Emotional trigger pattern
- Audience relevance cues

### Breakdown

- Clear component breakdown of structure
- Practical interpretation for creators

### Key Takeaways

- Actionable lessons a creator can apply immediately

### Create My Version

- Direction for producing an original version, not a copy
- Angle differentiation guidance

### Suggested Original Ideas

- Multiple idea directions based on the analysis
- Originality-first suggestions

### Continue to Content Creator

- Explicit handoff step to Content Creator

## Explicit Exclusions

Do not include in Viral Scanner v1.0:

- direct publishing workflow
- asset generation or rendering
- video editing features
- timeline editing
- production package generation
- AI voice/music generation

## UX Principles

- One step, one purpose, one primary action
- User should always know what to do next
- Keep flow fast, clear, and low-friction
- Favor structured outputs over long unstructured text

## Sprint 1 Scope

Sprint 1 is foundation only:

- lock product definition and boundaries
- lock workflow sequence
- provide UX-ready frontend scaffold
- avoid full backend implementation

## Product Vision Fit

Gatekeeper AI remains an AI Content Orchestrator.

Viral Scanner contributes by structuring viral analysis into reusable creative direction and then handing off to Content Creator.
